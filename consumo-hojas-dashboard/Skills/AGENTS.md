# AGENTS.md

Reglas locales para `Skills/`.

- Toda skill activa vive en `Skills/<skill-name>/SKILL.md`.
- El frontmatter debe pasar `./scripts/validate-skills.sh`.
- Usar siempre `Skills/` con mayuscula en docs.
- Si creas o cambias una skill, sincronizar `README.md`, `AGENTS.md` y referencias locales afectadas.
- Mantener skills cortas, accionables y atadas a contratos reales del repo.
